 Superheroes API

A Flask REST API for tracking heroes and their superpowers.
This API allows users to view heroes, powers, and assign powers to heroes with validation rules.

 Features

View all heroes

View a single hero and their powers

View all powers

Update an existing power

Assign powers to heroes

Validations for data integrity

JSON responses with proper HTTP status codes

 Tech Stack

Python 3.8+

Flask

Flask SQLAlchemy

Flask Migrate

SQLite

SQLAlchemy Serializer

Postman (for testing)
